An identity map game in two phases

How to play
  1. Phase 1 — Think Like Christopher
    12 rounds. Each round shows a scene from the novel and asks what Christopher would do. Choose the correct answer — the one that reflects how Christopher actually thinks. Wrong answers cost you a life.
  2. 3 lives
    You start with 3 hearts. Lose one for every wrong answer. Lose all three and the game ends.
  3. Watch the identity map build
    Every answer adds a node to Christopher's identity map on the right — green for correct, red for wrong.
  4. Phase 2 — Build Your Own Map
    After Phase 1, you answer the same style of questions about yourself. No wrong answers. Your identity map is built side by side with Christopher's so you can compare.
  5. The goal
    Understand how Christopher sees the world — and discover how your own identity compares to his.
